== Description ==
A free add-on for [Caldera Forms](https://wordpress.org/plugins/caldera-forms/). A simple, form processor which can send form data to Mautic using Mautic's REST API.

= How Does It Work =
Mautic Configuration:
1. Go To Mautic >> Configuration >> API Settings
2. Set "API enabled?" to "Yes"
3. Set "Enable HTTP basic auth?" to "Yes"
4. Create a new Mautic user with access to add and update new contacts

WordPress Plugin Configuration
1. Activate "Caldera Forms - Mautic Integration"
2. Create a new Caldera Form
3. Go to Processors in that Form
4. Click on "Add Processor"
5. Select Mautic Integration by NetTantra
6. Set "Mautic Base URL" to the URL of your mautic installation
7. Set "Mautic Username" to the username of the newly created api user
8. Set "Mautic Password" to the password of the newly created api user
9. Setup the fields as necessary using the Mautic Field machine names as "Mautic Field Aliases" and Caldera Field Shortcodes as "Values to Send"
